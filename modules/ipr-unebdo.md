# IPR-UNEBDO
Opposability and Evidence Module

## Function
Provides technical opponibility and verifiable evidence for all actions linked to an IPR.

## Capabilities
- hash-based evidence
- audit reproducibility
- long-term verification
- independent validation
- time-opposable trace

## Role in the stack
Foundation layer of the IPR system.
Without UNEBDO, no strong technical opponibility exists.

## Operational impact
Transforms an action into a verifiable and opposable event.
